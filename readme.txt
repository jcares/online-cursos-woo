== Descripción ==

Plataforma de Cursos Online PCCURICO es un tema de WordPress optimizado para crear sistemas de aprendizaje en línea, academias digitales y plataformas educativas.

Este tema ha sido optimizado para funcionar con WooCommerce y sistemas LMS, permitiendo la venta y gestión de cursos online de forma profesional.

Su estructura está diseñada para instituciones educativas, centros de capacitación, OTEC, academias profesionales y organizaciones que necesiten ofrecer cursos a través de internet.

El tema ofrece una interfaz moderna, limpia y rápida, compatible con el editor de bloques de WordPress y completamente adaptable a dispositivos móviles.

Características principales:

- Optimizado para venta de cursos con WooCommerce
- Compatible con Tutor LMS y Edwiser Bridge
- Diseño limpio y profesional
- Compatible con Gutenberg
- Adaptable a dispositivos móviles
- Optimizado para SEO
- Código ligero y seguro
- Soporte para múltiples idiomas
- Compatible con RTL
- Alto rendimiento y carga rápida

Ideal para:

- Plataformas de cursos online
- Instituciones educativas
- Centros de formación profesional
- Academias digitales
- Programas de certificación
- Capacitación empresarial

== Instalación ==

1. En el panel de administración de WordPress, ir a Apariencia > Temas.
2. Hacer clic en "Añadir nuevo".
3. Seleccionar "Subir tema".
4. Cargar el archivo .zip del tema.
5. Hacer clic en "Instalar ahora".
6. Activar el tema.

Una vez activado se recomienda instalar los siguientes complementos:

- WooCommerce
- Tutor LMS
- Elementor (opcional)
- Yoast SEO
- WP Mail SMTP
